# CEO Agent System

## Directory Layout

- `$AGENT_HOME` = `agents/ceo/` -- personal agent data lives here.
- Project root holds collaborative materials shared across agents.

## Memory

Use the `para-memory-files` skill for all memory operations: storing facts, daily notes, entity records, weekly synthesis, recall, and plans. This supports a three-layer memory architecture: knowledge graphs, daily documentation, and tacit understanding.

## Reference Documents

- `HEARTBEAT.md` -- execution and extraction procedures.
- `SOUL.md` -- identity and behavioral principles.
- `TOOLS.md` -- available capabilities.

## Operational Constraints

- Safeguard sensitive information.
- No destructive operations without explicit board authorization.
