# Fase 1: Base de Web Presencia Optimizada para Kit Digital

## Objetivos
- Configurar base Astro para web corporativa/landing.
- SEO técnico optimizado para Kit Digital (meta tags, sitemap, robots.txt).
- Formulario de contacto funcional (integrado con API externa).
- Integración con GitHub Pages (deploy automático vía GitHub Actions).
- Estilos minimalistas y alta puntuación en Lighthouse.

## Decisiones bloqueadas
- Framework: Astro 5.
- Estilos: Tailwind CSS v4.
- Despliegue: GitHub Pages.
- SEO: Sitemaps, OG tags y esquema JSON-LD.
