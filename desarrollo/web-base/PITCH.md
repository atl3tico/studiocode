# Oferta: Web Negocio (Sistema de Gestión)
"Deja de perder dinero con citas olvidadas y presupuestos sin seguimiento. No te vendemos una web, te entregamos tu propio sistema de gestión de clientes (CRM) que trabaja para ti.

¿Qué incluye este pack?
1. **Gestión inteligente de citas:** Tus clientes reservan desde la web y se sincroniza automáticamente con tu Google Calendar o iCalendar.
2. **Recordatorios automáticos:** El sistema envía emails bonitos y recordatorios para reducir drásticamente las cancelaciones (no-shows).
3. **Presupuestos profesionales:** Envía presupuestos que el cliente puede aceptar con un clic.
4. **CRM integrado:** Ten el historial completo de cada cliente: qué servicios contrataron, qué presupuestos han recibido y cuándo les toca el siguiente mantenimiento.
5. **Visibilidad 360:** Todo lo que tu negocio necesita para pasar de ser un "trabajador autónomo" a ser una empresa organizada y eficiente.

Tecnología de nivel empresarial, financiada al 100% por el Kit Digital."
