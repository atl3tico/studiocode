# Checklist Kit Digital — Web Negocio

> Referencia: Bases reguladoras Kit Digital (BOE 2022 y actualizaciones 2024).
> Solución aplicable: **Sitio Web y Presencia en Internet** + **Gestión de Clientes**.

## Segmentos elegibles
| Segmento | Empleados | Ayuda máxima |
|----------|-----------|--------------|
| I        | 10–50     | 12.000€      |
| II       | 3–9       | 6.000€       |
| III      | 0–2       | 2.000€       |

## Requisitos obligatorios
- [ ] **Dominio propio** registrado a nombre del beneficiario (mínimo 12 meses).
- [ ] **Hosting** con disponibilidad ≥ 99% (mínimo 12 meses).
- [ ] **Diseño responsive** (móvil, tablet, escritorio).
- [ ] **Accesibilidad** WCAG 2.1 nivel AA.
- [ ] **SEO básico**: title, meta description, H1 único, sitemap.xml, robots.txt.
- [ ] **RGPD**: Política de privacidad, Aviso legal, banner cookies (IAB TCF v2.2).
- [ ] **Formulario de contacto / captura de leads** con confirmación por email.
- [ ] **Analítica**: integración GA4 o Plausible con consentimiento.
- [ ] **Panel de gestión** accesible para el cliente (documentado).

## Requisitos opcionales (diferenciadores)
- [ ] Sistema de citas online con recordatorios automáticos.
- [ ] Integración CRM (pipeline de leads con estado: Nuevo / Contactado / Cerrado).
- [ ] Facturación / presupuestos automatizados.
- [ ] Chat en tiempo real (Crisp / Tawk.to).

## Documentación entregable al cliente
- [ ] Manual de usuario del panel CRM (PDF / Notion).
- [ ] Credenciales de acceso (entregadas por canal seguro).
- [ ] Certificado de despliegue (URL + SSL verificado).
- [ ] Informe SEO inicial (Lighthouse ≥ 80 en Performance, Accessibility, SEO).
